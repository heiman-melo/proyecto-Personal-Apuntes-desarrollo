algunos o principales de componentes material ui

1) '<'Container maxWidth="xs" sx={{ border: 2, boxShadow: 1, pb: 2,}}> '<'/Container>
centra su contenido horizontalmente

2) '<'Typography variant="h1" component="h2" mb={2} color="primary" align="center" boxShadow={2} pb={2}>Soy un h1 '<'/Typography>
component = h2 esta sera la etiqueta , Este componente es compatible con todos system props,
variant = tendra estilos para que parezca un h6 

3) <'Box sx={{ border: 2, borderColor: "peru", p: 2, bgcolor: "#111", color: "white",}}> 
sirve como un componente contenedor
