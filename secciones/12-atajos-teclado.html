Atajos del teclado

1. Manipulación de Líneas (El "Ninja" del código)
Estos te ahorran segundos valiosos de "copiar y pegar" manualmente.

Mover una línea arriba/abajo: Alt + ↑ / ↓

Uso: Reordena funciones o variables sin romper el flujo.

Duplicar línea: Shift + Alt + ↑ / ↓

Uso: Perfecto cuando necesitas crear una variable similar a la anterior.

Eliminar línea: Ctrl + Shift + K (en VS Code) o Ctrl + Y (en otros).

Comentar: Ctrl + k + c

Descomentar: Ctrl + k + u

Uso: Para probar cambios rápido sin borrar código.

2. Multitarea y selección (El poder del "Multi-Cursor")
Esta es la habilidad que más tiempo ahorra.

Selección múltiple (Cursor en varios lados): Alt + Click

Uso: Escribe en 5 líneas al mismo tiempo.

Seleccionar la siguiente ocurrencia: Ctrl + D

Uso: Selecciona una palabra y presiona esto para seleccionar la siguiente igual. Ideal para renombrar variables locales rápido.

Seleccionar todas las ocurrencias: Ctrl + Shift + L || ctrl + Shift + h 

Uso: Cambia el nombre de una clase en todo el archivo de un solo golpe.

3. Navegación Veloz (No más scroll infinito)
Deja de usar la rueda del ratón para buscar archivos.

Ir a un archivo: Ctrl + P

Uso: Escribe el nombre del archivo y presiona Enter. Es instantáneo.

Ir a una línea específica: Ctrl + G

Uso: Útil cuando la consola te dice: "Error en la línea 154".

Ir a la definición: F12

Uso: Salta directamente a donde se creó la función o clase que estás usando.

Ver definición sin saltar: Alt + F12

Uso: Abre una pequeña ventana flotante para leer el código de la función sin irte de donde estás.

4. Gestión del Entorno (Control de la "Nave")
Abrir/Cerrar la Terminal: Ctrl + ` (la tecla de la tilde invertida).

Dividir la pantalla (Split View): Ctrl + \ (VS Code).

Uso: Ver el HTML en un lado y el CSS en otro.

Cerrar pestaña actual: Ctrl + W

Reabrir pestaña cerrada por error: Ctrl + Shift + T

5. El Santo Grial: Emmet
Si haces Web (HTML/CSS), Emmet no es un atajo, es una necesidad. Viene integrado en VS Code.

Ejemplo: Escribir ul>li*5>a y presionar Tab.

Resultado: Te crea una lista con 5 elementos y enlaces internos automáticamente.
